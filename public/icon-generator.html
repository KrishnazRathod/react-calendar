PWA Icon Generator

Generate icons for your Progressive Web App

Instructions:

  1. Click "Generate 192x192" to create the small icon
  2. Right-click the canvas and "Save image as..." → Save as icon-192.png in the public folder
  3. Click "Generate 512x512" to create the large icon
  4. Right-click the canvas and "Save image as..." → Save as icon-512.png in the public folder
  5. Alternatively, use an online tool like RealFaviconGenerator or PWA Builder
